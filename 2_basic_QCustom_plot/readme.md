# Basic plotting with QCustomPlot

## add QCustomPlot .cpp and .h
Project >> Add Existing Files...
    
    qcustomplot.cpp
    qcustomplot.h

## .ui
Graphics View >> Promote to ...

    Promoted class name = QCustomPlot
    Header file = qcustomplot.h

## mainwindow.cpp
Use the pointer to call methods:

    ui->customPlot
